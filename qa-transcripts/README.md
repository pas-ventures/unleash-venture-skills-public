# Unleash Q&A Transcripts

Anonymised transcripts of Unleash Community Founder Q&A sessions. Members can read, search, and query these with their own AI tools (Claude, ChatGPT, etc.) for reference and pattern-matching across sessions.

## What's here

Each transcript is a single Markdown file, one per session:

```
unleash-qa-N-<topic-slug>.md
```

Example: `unleash-qa-5-early-stage-validation.md`

## Anonymisation policy

Per the policy locked 2026-04-23:

- **Names**: fully replaced with roles — `Host`, `Community Manager`, `Founder 1`, `Founder 2`, …
- **Business specifics** (company names, customer names, specific deal terms, revenue figures tied to a specific venture): best-effort scrub. If you spot identifying detail that shouldn't be public, flag to the Unleash team.
- **Session recording** itself is stored separately in the Members Area and **must not be shared externally**. Only these anonymised transcripts + the post-session PDFs are safe to pass around inside the community.

## How to use these

Drop one into a Claude / ChatGPT conversation and ask questions like:

- *"What did founders in this session say about pricing discovery?"*
- *"Which questions did the Host challenge most sharply — why?"*
- *"Summarise the top 3 validation mistakes mentioned across the Q&A."*
- *"Draft a customer-interview script using the Mom Test principles that came up here."*

For deeper pattern-matching across sessions, load multiple transcripts into the same conversation.
